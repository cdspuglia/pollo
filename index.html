What's Pollo?

Pollo is a PHP, event sourced, domain driven designed polling system. With Pollo, you can create and publish polls with two or more options ready to be voted. Simple as that.

Why Pollo?

At Club degli Sviluppatori Puglia, we host a tech talk every month and we like to democratically decide which one will take place by voting on several proposals by our members.

Event sourcing? Domain driven design?

Pollo is built using concepts from domain driven design, CQRS and event sourcing. We could have done this more straight and simple, but Pollo has been built also for tutoring and learning purposes and that's why we chose such more advanced developing techniques.

How do I install it?

Pollo runs on a classic PHP environment, no special requirements for now. To install Pollo, just run read the Getting started section of the Development guidelines page.
